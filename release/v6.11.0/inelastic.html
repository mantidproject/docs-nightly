New Features¶

  • Added the ability to load A0 fit parameter data into the Function (Q) tab of the QENS Fitting interface.

  • Add option to not delete subtracted workspaces when adding new data in the Container Subtraction tab of the Corrections interface.

  • Add Output Composite Members option to I(Q, t) tab in Inelastic QENS Fitting

  • Added an option to “Tie Amplitudes” for two Lorentzians in the Convolution tab of the Inelastic QENS Fitting interface.

  • It is possible to access the Slice Viewer or 3D Plot from the OutputPlot widget of the in Elwin Tab of Data Processor Interface for output workspaces containing more than 1 histogram.

  • Rename tabs of QENS Fitting from MSD Fit, I(Q,t) Fit, ConvFit and F(Q) to MSD, I(Q,t), Convolution and Function(Q) respectively. Add tooltip.

  • Add Load Workspace History option to the settings dialog for Indirect/Inelastic interfaces.

Bugfixes¶

  • The dialog window for adding data in the Elwin Tab of the Data Processor Interface no longer freezes when adding data.

  • Fixed a crash on the Quasi tab of the Inelastic Bayes Fitting interface caused by attempting to load a WorkspaceGroup rather than the expected Workspace2D.

  • Prevented a crash on the Quasi tab of the Inelastic Bayes Fitting interface caused by clicking Run before data has finished loading.

  • Fixed a bug in the Monte Carlo error calculation on the I(Q, t) tab of the Data Processor Interface where the first bin had an error of zero.

  • Fix a bug in which sliders didn’t respond to changes in Emin and EMax properties when changed from the property browser in the Moments tab of the Data Processor interface.

  • Fix a freeze in Elwin Tab of Data Processor Interface that happens when running the tab.
