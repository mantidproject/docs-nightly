Mantid 6.13.1 Release Notes¶

Table of Contents

  • Citation

  • Changes in this version

This is a patch release that corrects some significant issues since version 6.13.0.

The changes are:

  • MantidWorkbench full installers will now work correctly on all Linux distributions.

  • Muon Analysis and Frequency Domain Analysis interfaces will now open on all Linux distributions.

  • SaveNXSPE will no longer throw an exception when creating nested groups with the same name.

Please cite any usage of Mantid as follows:
